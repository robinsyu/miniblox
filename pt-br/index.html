Perguntas Frequentes

O MiniBlox é gratuito?

Sim! O MiniBlox é completamente gratuito para jogar diretamente no seu navegador sem downloads ou instalações.

Posso jogar MiniBlox em dispositivos móveis?

Sim, o MiniBlox foi projetado para ser responsivo e funciona bem tanto em desktop quanto em dispositivos móveis com controles de toque.

Preciso criar uma conta?

Não é necessária uma conta para começar a jogar, mas criar uma permite salvar seu progresso e acessar recursos adicionais.

Posso jogar com meus amigos?

Sim! O MiniBlox suporta funcionalidade multijogador para que você possa construir e explorar com amigos em mundos compartilhados.

Existem atualizações regulares?

Sim, atualizamos regularmente o MiniBlox com novos recursos, blocos e modos de jogo para manter a experiência sempre nova e emocionante.
